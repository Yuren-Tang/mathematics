## 3. Six controlling B1 corrections

1. A fixed support-coordinate inverse image is even; a fixed root-label inverse image is a matching.
2. Bare matching plus four-flow is insufficient without $(B,D)$ or component endpoint-parity/$T$-join data.
3. For fixed $g$, $T_g^{(1)}\to\mathscr A_5$ means componentwise compression of the same embedding.
4. An external cover integrates on a prescribed dual only after all dual cycle holonomies vanish.
5. $J_g\to\mathscr A_5$ is only the old-colour-factorable subclass.
6. Singleton words require an odd translation when $\mathscr A_5$ denotes the even component.

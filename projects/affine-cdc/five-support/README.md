## Scope

The five-support programme strengthens, but does not replace, the complete paper-level Cycle Double Cover spine in the parent project. No global five-support theorem is claimed.

The current controlling fixed-lift criterion is

$$
T_g^{(1)}\longrightarrow\mathscr A_5,
$$

not merely the global-colour quotient criterion $J_g\to\mathscr A_5$.

The current sharp frontier is the graph-level localisation and composition of full-rank route-locked defects, together with the associated four-pole and horizontal bad-flow transfer problems.

## 1. Root-flow channel graphs

Let

\[
\lambda:E(G)\to R_5
\]

be a root-valued flow, with indexed even supports

\[
F_1,\ldots,F_5.
\]

For a root

\[
s=pq\in R_5,
\]

define the channel subgraph

\[
A_s(\lambda)
=
\{h\in E(G):\lambda(h)+s\in R_5\}.
\]

### Theorem 1.1 — support-pair identity

\[
\boxed{A_{pq}(\lambda)=F_p\triangle F_q.}
\]

### Proof

For a root label `x`, the sum `x+pq` is a root exactly when `x` contains exactly one of `p,q`. This is exactly membership in one, but not both, of `F_p,F_q`. ∎

Since `F_p,F_q` are even, `A_s` is even. In a cubic graph every nonempty connected component of `A_s` is a source circuit.

## 2. Exact root correction on one channel circuit

Let `z` be a binary cycle word supported in `A_s`. Then

\[
\lambda^{s,z}:=\lambda+s z
\]

is again root-valued.

Conversely, if a root-valued flow agrees with `lambda` off one connected cycle and differs by the constant root direction `s` on that cycle, the cycle is one component of `A_s`.

Thus root-direction cycle corrections are exactly ordinary support-pair Kempe switches.

## 3. Two-edge evaluation

Fix two distinct marked edges `e,f`. Define

\[
\operatorname{ev}_{e,f}:\mathcal C(A_s)\to\mathbf F_2^2,
\qquad
z\longmapsto(z_e,z_f).
\]

Because every component of `A_s` is a circuit, its image is determined by the channel-component positions of the two marks.

### Theorem 3.1 — evaluation image table

Assume both `e,f` lie in `A_s`.

1. If they lie in different circuit components, then
   \[
   \operatorname{im}\operatorname{ev}_{e,f}=\mathbf F_2^2.
   \]
2. If they lie in the same circuit component, then
   \[
   \operatorname{im}\operatorname{ev}_{e,f}
   =\{(0,0),(1,1)\}.
   \]

If exactly one marked edge lies in `A_s`, the image on that coordinate is `{0,1}` and the other coordinate is frozen. If neither lies in `A_s`, the image is zero.

### Proof

A binary cycle word in a cubic even subgraph is a union of complete circuit components. Therefore each component contributes one independent indicator bit. ∎

### Dual formulation

The annihilator of the evaluation image consists of the two-mark vectors whose extension belongs to the cut space of `A_s`.

In the same-component case the nonzero annihilator is generated by `(1,1)`: every channel cycle meets `{e,f}` evenly. This is the exact two-edge channel stress.

## 4. Equality repair channels

Assume

\[
\lambda(e)=\lambda(f)=a.
\]

A root direction `s` repairs one marked edge while preserving root-valuedness exactly when `s` intersects `a` in one support index. There are six such roots.

For every such `s`, both marked edges lie in `A_s`. If a channel circuit contains `e` but not `f`, switching it changes

\[
a\longmapsto a+s,
\]

while leaving the other label equal to `a`. The two labels are then distinct intersecting roots.

### Theorem 4.1 — equality lock criterion

A given equality pair is not repairable by any one root-Kempe switch if and only if, for all six roots `s` intersecting `a`, the marked edges `e,f` lie in the same circuit component of

\[
A_s=F_p\triangle F_q.
\]

Equivalently, all six evaluation images are the diagonal line

\[
\langle(1,1)\rangle.
\]

This is precisely the six-channel equality full lock.

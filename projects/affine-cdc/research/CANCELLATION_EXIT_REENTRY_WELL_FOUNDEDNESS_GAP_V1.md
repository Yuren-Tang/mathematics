## 1. The three distinct objects at a cancellation

Let

\[
D^+\longrightarrow D^-
\]

be a forward equal-face `2--0` cancellation. The graph `D^+` contains two equal root triangles joined by one edge; `D^-` reconnects the two remaining pairs and has two fewer vertices.

Three logically different statements must not be conflated.

### A. Current-flow forward cancellation

The particular current root flow on `D^+` descends to a particular inherited root flow on `D^-`.

For that same inherited flow, reversing the cancellation with the stored equal-face labels succeeds. This is ordinary reversibility of the selected current-flow move.

### B. Lower-order existence

An induction or minimal-counterexample hypothesis says only:

\[
\exists\rho^-:E(D^-)\to R_5.
\]

The returned flow `rho^-` need not be the inherited current flow of A.

### C. Contextual inverse expansion

To recover a flow on `D^+`, let the two reconnected edges of `D^-` carry roots

\[
a,b.
\]

The restored central value is

\[
x=a+b.
\]

The inverse succeeds root-valuedly only if `a,b` are distinct and intersect. If

\[
a=b,
\]

then `x=0` and the restored boundary is quadruple equality. If `a,b` are disjoint, `x` is a co-root and the restored boundary is doubled-disjoint.

Thus B does not imply C.

## 2. Exact local obstruction to treating cancellation as an exit

### Theorem 2.1 — inverse-cancellation trichotomy

For an arbitrary lower-order root flow on `D^-`, inverse equal-face insertion gives exactly:

\[
\begin{array}{c|c}
\text{relation of }a,b&\text{outcome on }D^+\\
\hline
\text{distinct intersecting}&\text{root success}\\
a=b&\text{one zero atom; boundary }a,a,a,a\\
a\cap b=\varnothing&\text{one co-root atom; boundary }a,a,b,b.
\end{array}
\]

Consequently:

\[
\boxed{
D^-\text{ root-soluble}
\not\Rightarrow
\text{the cancelled pair reinserts root-valuedly}.}
\]

This is not a hypothetical new coefficient type. It is exactly the already proved first-failure grammar.

## 6. Route and direct terminals do not silently preserve the constructing flow

A tempting repair is to retain the particular current flow which generated the forward history, because every selected flip/cancellation is then reversibly labelled.

This works only if the cap-compatible terminal uses the same descendant flow.

### Route terminal

A route terminal generally applies a Kempe component switch to move the boundary state into `K_i`. That switch can change internal edge labels on the descendant. The resulting cap-compatible flow is not automatically the original current flow transported unchanged through the history.

### Direct theta terminal

At a zero-vertex direct matching, the terminal theorem explicitly chooses an arbitrary root triangle on a theta graph (or arbitrary distinct intersecting loop roots before the zero-to-root resolution). These values need not agree with the inherited singular boundary values of the constructing flow.

Therefore the current-flow reversibility observation does not presently eliminate cover-independent transfer.

## 7. Three logically sufficient repair routes

The global candidate can be repaired by any one of the following, with all hypotheses stated explicitly.

### Repair A — history-coherent terminal selection

Prove that every forward singular-carrier history has a cap-compatible terminal root flow whose restriction to every retained source edge is history-coherent, so every inverse flip and inverse cancellation succeeds with the stored labels.

This would remove first-failure confluence entirely for that selected terminal.

It must cover both route-changing and zero-vertex direct terminals; existence of an arbitrary cap-compatible flow is insufficient.

### Repair B — cancellation macro-return with a strict secondary rank

Extend the contextual state space to include the lower-order subcall and every possible root/token re-entry after inverse insertion. Prove that every cancel--solve--reinsert macro-transition either succeeds or returns to an order-`N` state with a strictly smaller independently defined rank.

Possible ranks requiring proof include:

- a strict causal-history prefix;
- a strictly enclosed carrier interval;
- a singular Morse/context complexity preserved across the recursive return;
- a finite game rank on the complete recursive state graph.

No such rank is presently established for arbitrary returned root flows.

### Repair C — inverse-weld flow-selection theorem

For every equal-face cancellation generated by the source movie, prove the existence of a lower-order root flow in which the two reconnected edges are distinct intersecting roots.

Equivalently, prove a boundary-state selection theorem ensuring that at least one lower-order flow lies in the root-success row of Theorem 2.1.

This is a specialized relative colored-weld extension theorem. It must not assume the very order-`N` cap-extension result being proved.

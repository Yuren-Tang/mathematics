## 1. Purpose

The two BBD Type H triangle shores have sixteen support monodromies and sixty-four raw ordered lifts.  The previous packets compressed their boundary data to four binary coordinates

\[
(\nu,\rho,\sigma,\tau)\in F_2^4.
\]

This packet proves a stronger global statement.

1. The sixteen boundary monodromies have an exact left--right bitorsor normal form controlled by two fixed transpositions.
2. Their fixed coefficient modules form a rectangular two-channel system.
3. The physical affine holonomy group has no nonzero pure translation kernel.
4. Since \(H^1(S_5,E_5)=0\), all BBD holonomies have a unique common ambient fixed origin.
5. The common shifted flow is either root-valued, giving Tait escape, or has a canonical zero/co-root defect network.
6. The nonzero coefficient space \(E_5\setminus\{0\}\) is naturally the edge set of \(K_6\); cubic conservation is exactly a triangle-or-one-factor law.
7. In the absence of zero edges, the co-root defects form disjoint internal paths and cycles whose endpoints are precisely Berge--Fulkerson one-factor vertices.

Thus the BBD branch no longer has an ambient translation alternative.  Its unique residual mechanism is a canonical defect-strip geometry.

No five-cycle double cover theorem, Berge--Fulkerson theorem, or strict replacement theorem is claimed.

## 2. Boundary bitorsor normal form

Let

\[
a=(12),\qquad b=(34),\qquad c=(23)(45)\in S_5.
\]

For the canonical BBD triangle, let

\[
\pi_{\nu\rho\sigma\tau}
\]

denote the monodromy indexed by the four quotient coordinates.

### Computational Theorem 2.1 -- exact normal form

For all four binary coordinates,

\[
\boxed{
\pi_{\nu\rho\sigma\tau}
=
a^\nu\,
\bigl(a^\sigma b^\tau c b^\tau a^\sigma\bigr)
\,b^\rho.
}
\]

Equivalently, if

\[
\pi_{00\sigma\tau}=a^\sigma b^\tau c b^\tau a^\sigma,
\]

then

\[
\boxed{
\pi_{\nu\rho\sigma\tau}
=a^\nu\pi_{00\sigma\tau}b^\rho.
}
\]

Exact enumeration of all sixty-four raw choices verifies the identity.

### Corollary 2.2 -- flat parity rectangles

For fixed \((\sigma,\tau)\),

\[
\pi_{10}=a\pi_{00},
\qquad
\pi_{11}=a\pi_{01},
\]

and

\[
\pi_{01}=\pi_{00}b,
\qquad
\pi_{11}=\pi_{10}b.
\]

Therefore

\[
\boxed{
\pi_{11}=\pi_{10}\pi_{00}^{-1}\pi_{01}.
}
\]

The parity square is a genuine group-torsor parallelogram: changing \(\nu\) is left multiplication by the fixed involution \(a\), while changing \(\rho\) is right multiplication by the fixed involution \(b\).

This is why the four monodromies in one cell are not a one-sided \(V_4\)-coset: the correct structure is a left--right bitorsor rectangle.

## 3. Rectangular norm-channel theorem

Define four coefficient vectors

\[
A_0=01111,\qquad A_1=10111,
\]

and

\[
B_0=00011,\qquad B_1=00101.
\]

The vectors \(A_\sigma\) are weight-four co-roots.  The vectors \(B_\tau\) are roots.

### Computational Theorem 3.1 -- fixed modules

For every coordinate quadruple,

\[
\boxed{
E_5^{\pi_{\nu\rho\sigma\tau}}
=
(1-\nu)\langle A_\sigma\rangle
\oplus
(1-\rho)\langle B_\tau\rangle.
}
\]

Thus:

\[
\begin{array}{c|c}
(\nu,\rho)&E_5^\pi\\
\hline
(0,0)&\langle A_\sigma,B_\tau\rangle\\
(1,0)&\langle B_\tau\rangle\\
(0,1)&\langle A_\sigma\rangle\\
(1,1)&0.
\end{array}
\]

The formula has a direct cycle interpretation.

- Type \(2^2 1\): \(A_\sigma\) is the characteristic vector of the four moved supports, and \(B_\tau\) is one transposed support pair.
- Type \(3\,2\): only the transposed-pair root \(B_\tau\) remains fixed.
- Type \(4\,1\): only the four-cycle vector \(A_\sigma\) remains fixed.
- Type \(5\): no nonzero even vector is fixed.

### Corollary 3.2 -- selector/orientation dual role

The residual coordinates change meaning when a channel is closed.

- In type \(2^2 1\), \(\sigma\) and \(\tau\) select the two fixed channels.
- In type \(3\,2\), \(\tau\) selects the fixed two-cycle, while \(\sigma\) reverses the three-cycle orientation.
- In type \(4\,1\), \(\sigma\) selects the fixed support, while \(\tau\) reverses the four-cycle orientation.
- In type \(5\), both residual bits are pure transport data.

## 5. Root rigidity eliminates the translation kernel

Let \(x\) be the initial indexed root cover.  Since every element of \(T\) is represented by a concatenation of physical based loops,

\[
\boxed{
x+v\text{ is root-valued for every }v\in T.
}
\]

For an edge \(e\), define the evaluation image

\[
T_e=\{v(e):v\in T\}\le E_5.
\]

Because \(T\) is \(S_5\)-stable and evaluation commutes with the coefficient action, \(T_e\) is an \(S_5\)-submodule of \(E_5\).

### Lemma 5.1 -- irreducibility of \(E_5\)

The natural even module

\[
E_5=\{w\in F_2^5:\sum_iw_i=0\}
\]

is irreducible under \(S_5\).

#### Proof

A nonzero vector has weight two or four.

- The orbit of a weight-two vector is the set of ten roots, which spans \(E_5\).
- If a submodule contains a weight-four vector, it contains its full co-root orbit.  The sum of two distinct co-roots is a root, so the submodule again contains the root orbit and hence all of \(E_5\).  ∎

Thus each \(T_e\) is either \(0\) or \(E_5\).

### Theorem 5.2 -- no nonzero physical translation kernel

\[
\boxed{T=0.}
\]

#### Proof

For every edge,

\[
x(e)+T_e\subseteq R_5
\]

because all translated covers are root-valued.  If \(T_e=E_5\), then

\[
x(e)+T_e=E_5,
\]

which contains zero and co-roots and therefore is not contained in the ten-root orbit \(R_5\).  Hence \(T_e=0\) for every edge.  Every translation tensor vanishes edgewise, so \(T=0\).  ∎

### Consequences

1. Two physical BBD loops with the same support monodromy have the same affine holonomy.
2. The affine holonomy descends from the sixty-four raw lifts to the sixteen monodromies.
3. Every parity plaquette is affine-flat.
4. Every individual cyclic norm defect vanishes.
5. Every boundary relation lifts without a residual pure translation.

The entire BBD ambient-translation branch is eliminated.

## 6. Vanishing of \(H^1(S_5,M)\)

Since \(T=0\), the projection

\[
\mathcal H\xrightarrow{\sim}S_5
\]

is an isomorphism.  Hence \(\mathcal H\) is the graph of a one-cocycle

\[
c:S_5\to M.
\]

### Theorem 6.1 -- cohomology vanishing

\[
\boxed{H^1(S_5,E_5)=0.}
\]

#### Proof

Let \(V=F_2^5\) be the permutation module.  Since five is odd,

\[
V=\mathbf1\oplus E_5.
\]

Moreover

\[
V\cong \operatorname{Ind}_{S_4}^{S_5}\mathbf1.
\]

By Shapiro's lemma,

\[
H^1(S_5,V)
\cong
H^1(S_4,F_2)
\cong F_2,
\]

because the abelianisation of \(S_4\) is \(C_2\).  Also

\[
H^1(S_5,\mathbf1)
=\operatorname{Hom}(S_5,F_2)
\cong F_2.
\]

Additivity on the direct sum gives

\[
H^1(S_5,V)
=
H^1(S_5,\mathbf1)
\oplus
H^1(S_5,E_5),
\]

so the final summand is zero.  ∎

Since the support action is trivial on the graph cycle factor,

\[
M\cong E_5^{\oplus\dim Z_1(P)},
\]

and therefore

\[
\boxed{H^1(S_5,M)=0.}
\]

Exact finite Wolfram computation using the Coxeter presentation independently finds sixteen cocycles and sixteen coboundaries on \(E_5\).

## 8. The natural K6 completion of E5

### Theorem 8.1 -- K6 edge model

\[
\boxed{
\Phi:E(K_6)\xrightarrow{\sim}E_5\setminus\{0\}.
}
\]

#### Proof

- If \(a,b\in[5]\), then \(q_a+q_b=e_a+e_b\), a weight-two root.  These are the ten edges of the induced \(K_5\).
- If one endpoint is \(\infty\), then \(\Phi(\{\infty,i\})=q_i\), a weight-four co-root.  These are the five edges of the star at \(\infty\).

The fifteen images are distinct and exhaust the fifteen nonzero vectors of \(E_5\).  ∎

### Theorem 8.2 -- triangle-or-one-factor law

Let \(u,v,w\in E_5\setminus\{0\}\) be distinct.  Under the K6 edge model,

\[
\boxed{
u+v+w=0
\iff
\text{the three K6 edges form either a triangle or a perfect matching.}
}
\]

#### Proof

The sum records the parity of the incidences of the six endpoint symbols, modulo the two relations generated by the isolated \(\infty\)-coordinate and the all-six vector.  A simple three-edge set has even endpoint parity exactly when it is a triangle, and has all six endpoints odd exactly when it is a perfect matching.  These are the only possibilities compatible with the handshaking parity.  ∎

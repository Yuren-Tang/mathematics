## 1. The remaining datum

After the binary four-port reduction, a minimal laminar interval `R` has:

- four ordered route ports;
- one locked matching;
- one internal closed scalar component `Z`;
- an external side-root signature `Sigma(R)`.

The first three items are finite and already classified. The only potentially unbounded datum is `Sigma(R)`.

The active corpus already supplies two mechanisms for such external semantics:

1. root-fibre local systems, whose failure is empty edge fibre, empty vertex relation, or component holonomy;
2. enriched co-root transport, whose state is a root-pair walk together with a side-root output word.

Hence the interval problem should be attacked by factorising `Sigma(R)` through these two existing systems rather than introducing a new routing automaton.

---

## 2. Root-fibre projection

For every external side attachment `e` of `R`, let `F_e` be the admissible root fibre determined by the two competing route witnesses on the four-port boundary.

At an internal attachment vertex, let `Rel_v` be the local zero-sum relation among the incident fibres.

### Target 2.1 — fibre trichotomy on the interval boundary

Prove one of:

1. some `F_e` is empty, giving a local defect edge;
2. every `F_e` is nonempty but some `Rel_v` is empty, giving a bounded defect vertex;
3. all local relations are nonempty and the interval side signature carries a global section up to one component holonomy class.

Cases 1 and 2 already descend to the defect-forest branch. Only Case 3 requires transport analysis.

---

## 3. Co-root transport projection

In Case 3, compare two local root sections along the interval. Their difference is root-supported unless the two repair roots are disjoint; in the disjoint branch the overlap carries one fixed co-root direction and an oriented octahedral transport word.

### Target 3.1 — bounded return extraction

Use the full-frame bound to locate a repeated frame within at most twenty-four transport vertices. The intervening segment has one of:

- identity transport;
- odd DDD return;
- nonreturn endpoint transport.

The odd return has a three-step DDD curvature core plus an identity remainder. The identity return lies in the affine `A3` translation kernel and is generated by bounded Coxeter and translation cells.

### Target 3.2 — interval compatibility

Show that a repeated-frame subsegment whose external attachments are contained in one minimal laminar interval can be replaced, split, or contracted without changing the four route ports, unless an attachment crosses the laminar order. A crossing attachment returns to the profile-escape branch.

## 4. Proposed descent measure

Refine interval complexity to

\[
\mathcal D(R)
=
\bigl(
 |\Sigma(R)|,
 N_{\mathrm{frame}}(R),
 |V(R)|,
 \beta(R),
 D(R)
\bigr),
\]

where:

- `|Sigma(R)|` is the number of external side attachments;
- `N_frame(R)` is the number of distinct full overlap frames realised by the side semantics;
- `D(R)` is the number of zero/co-root defect edges.

Use lexicographic order.

A successful local surgery should first lower the number of external attachments or distinct frames. Only if those are unchanged should vertex count, cycle rank, and defect count be used.

This ordering prevents a replacement from being declared progress merely because it shortens the visible state word while increasing the untracked side semantics.

---

## 5. Expected master lemma

### Target 5.1 — side-signature control lemma

For a support-minimal pair-preserving laminar four-port interval `R`, one of the following holds:

1. **fibre defect:** an empty edge fibre or vertex relation gives a smaller defect unit;
2. **route escape:** a crossing external attachment gives an admissible profile-changing switch, or one of the known DDD lift failures;
3. **bounded holonomy atom:** a repeated frame localises an odd DDD core or identity translation cell with no larger side signature;
4. **separator:** the signature splits across a cyclic cut/four-pole with the selected matching;
5. **serial contraction:** all side attachments are order-compatible and a minimal segment can be replaced with strictly smaller `D(R)`.

No sixth branch should be necessary.

---

## 6. Strategic assessment

The project is no longer blocked by lack of a finite obstruction alphabet. The alphabet is stable:

- root-fibre defect;
- DDD curvature;
- flat translation;
- four-pole separator;
- serial interval.

The remaining theorem is a localisation/composition theorem showing that arbitrary side semantics must enter one of these already known bins with strict descent.

This is the correct point at which proof-development support may later formalise a DAG, but the Research Lead must still establish the side-signature control lemma and its descent measure before the statement surface is stable.

---

## 7. Trust boundary

### Exact here

- identification of `Sigma(R)` as the only unbounded datum after four-port reduction;
- applicability of the existing root-fibre and enriched co-root transport languages;
- bounded full-frame and odd-return normal forms as finite diagnostics;
- a descent measure that prioritises external semantics over visible path length.

### Still open

- the side-signature control lemma itself;
- composition-safe replacement of repeated-frame subsegments;
- extraction of a separator from split signatures;
- serial contraction in the fully order-compatible case;
- global horizontal induction and the five-support theorem.
